# SEO アドバイス

コードやコンテンツの SEO 改善提案を、根拠と共に提供します。

**注意: このスキルはアドバイスのみを提供します。コードやファイルの変更は行いません。**

## アドバイス手順

### 1. 対象の確認

- ユーザーが指定したコード・コンテンツ・URL を確認する
- 対象の技術スタック・コンテキストを把握する
- アドバイスの焦点（全般 or 特定領域）を確認する

### 2. SEO 観点の分析

以下の観点で問題点・改善機会を分析:

- **メタデータ**: title, description の最適化余地
- **コンテンツ構造**: 見出し階層、内部リンク構造
- **技術的 SEO**: レンダリング、クロール効率
- **構造化データ**: 追加・改善の機会
- **パフォーマンス**: 速度に影響する明らかな問題
- **モバイル対応**: レスポンシブ・UX の問題
- **AI 検索対応**: AI Overview への最適化余地

### 3. 根拠の提示

各提案に対して:
- Google の公式ガイドライン・ドキュメントに基づく根拠
- SEO への影響度（高/中/低）
- 実装の難易度（高/中/低）

## 活用シーン

- コードレビュー前の SEO 観点チェック
- 新機能の設計段階での SEO 方針相談
- 既存ページの改善余地の洗い出し
- SEO 施策の優先順位付け

## 連携スキル

- アドバイスに基づく実装は `seo-implement` を使用してください
- 詳細なページ分析は `seo-page-audit` が有効です
