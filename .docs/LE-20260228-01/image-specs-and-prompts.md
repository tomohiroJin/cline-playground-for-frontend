# 迷宮の残響 — 画像アセット仕様 & ChatGPT プロンプト集

## 概要

本ゲームの画像アセットには**透過処理（アルファチャンネル）**が必須のものがある。
ChatGPT (DALL-E) は直接透過PNGを出力できないため、以下の手順で作成する：

1. ChatGPT で「透過風」画像を生成（背景を黒一色 or 特定色で統一）
2. 画像編集ツール（Photoshop, GIMP, remove.bg 等）で背景を除去し透過PNG化
3. WebP に変換（品質82%、アルファチャンネル保持）

## 後処理手順（透過が必要な画像）

### 方法1: GIMP（無料）
1. 画像を開く
2. `色` → `色を透明度に...` → 色: 黒 (#000000) を選択
3. 不透明度のしきい値を調整（10-20程度）
4. `ファイル` → `名前を付けてエクスポート` → `.webp` 形式、品質82%

### 方法2: ImageMagick（コマンドライン）
```bash
# 黒を透明に置換（fuzz=10%で近似色も含む）
convert input.webp -fuzz 10% -transparent black output.webp
```

### 方法3: Photoshop
1. 画像を開く
2. `選択範囲` → `色域指定` → 黒を選択（許容量10-20）
3. Delete キーで削除
4. WebP で書き出し（品質82%、透明度保持）

## プログラム的な透過処理（Pillow スクリプト）

AI画像生成ツールは透過画像を直接生成できないため、Python + Pillow によるプログラム処理で透過変換を行う。

### 処理スクリプト

`image-processing/process_images.py` に3種類の透過処理を実装：

| パターン | 対象 | 枚数 | 概要 |
|---------|------|------|------|
| 楕円グラデーションマスク | `le_overlay_*.webp` | 5枚 | 中央透明、端のエフェクト残す |
| 矩形ビネットマスク | `le_bg_*_near.webp` | 5枚 | 中央透明、フレーム的効果 |
| 輝度ベースアルファ | `le_title_mid.webp` | 1枚 | 暗部を透明に |

### 実行手順

```bash
# 透過処理実行
python3 .docs/LE-20260228-01/image-processing/process_images.py

# 処理結果検証
python3 .docs/LE-20260228-01/image-processing/verify_images.py
```

詳細な技術仕様は `image-processing/03-processing-specs.md` を参照。
