# IPNE ゲーム概要

## 基本情報

| 項目 | 内容 |
|------|------|
| タイトル（日本語） | 調査中につき脱出不能 |
| タイトル（英語） | Investigation in Progress: No Escape |
| 略称 | IPNE |
| ジャンル | 短時間・判断型・見下ろし迷路アクションゲーム |
| プラットフォーム | WEBブラウザ（PC／スマートフォン） |
| プレイ時間 | 1プレイ 5〜10分 |

---

## コンセプト

> 突如出現した不安定なダンジョンに調査員として侵入し、
> 自動生成される迷路を探索・判断しながら、
> ボスを撃破して鍵を入手し脱出する。

**操作は単純、判断は多い。**

- 1プレイ5〜10分
- 操作は極限まで単純化
- 「操作」ではなく「判断の質」が上達する体験を提供

---

## 目的と勝利条件

### クリア条件

1. ダンジョン内に存在する **ボスモンスターを撃破**
2. ボスがドロップする **鍵を入手**
3. 鍵を持って **ゴールに到達**

### ゲームオーバー条件

- プレイヤーのHPが0になる
- 死亡は明言せず「調査中断・行方不明」扱い
- 即リトライ可能

---

## ゲームの核

- 操作は単純
- 判断は多い
- 情報は行動の結果としてのみ得られる

**短時間で、判断が上達する迷路アクション**

---

## MVP ロードマップ

| MVP | 目的 | 主要機能 | 状態 |
|-----|------|----------|------|
| **MVP0** | 技術的E2E確認 | 固定迷路、移動、ゴール | 完了 |
| **MVP1** | 探索体験の検証 | 自動生成迷路、自動マッピング | 完了 |
| **MVP2** | 危険を伴う探索 | 敵AI、HP、戦闘、アイテム | 完了 |
| **MVP3** | 判断の重層化 | 職業差、成長システム、罠・壁ギミック | 完了 |
| **MVP4** | 完成形への仕上げ | タイマー、評価、チュートリアル、演出 | 完了 |
| **MVP5** | 音響演出 | 効果音、BGM、音量設定 | 完了 |

### MVP別の検証ポイント

#### MVP0: 歩けて脱出できる
- 操作は気持ちいいか
- 画面は見やすいか
- スマホ／PCで成立するか

#### MVP1: 迷路として成立する
- 迷うことが楽しいか
- マップを見る頻度
- 一本道3分が長いか短いか

#### MVP2: 判断が生まれる
- 「戦う／避ける／進む」の判断が発生するか
- 戦闘が面倒にならないか
- HP設計はジリ貧にならないか

#### MVP3: 完成形に近い手触り
- 毎回判断が変わるか
- 5〜10分に収まるか
- 盗賊と戦士の体験差が伝わるか

#### MVP4: 仕上げ・演出・リプレイ価値
- 初見でルールが理解できるか
- リプレイ動機が生まれるか
- 演出が気持ちよさに寄与するか

#### MVP5: 音響演出
- 効果音が気持ちよさに寄与するか
- BGMがゲーム体験を邪魔していないか
- 音量調整が直感的か
- iOS/モバイルでも音が再生されるか

## 評価システム（S/A/B/C/D）

クリアタイムによる5段階評価：

| 評価 | 条件 | エピローグ |
|------|------|------------|
| S | 0〜2分 | 迷宮の支配者 |
| A | 2〜3分 | 熟練の生還者 |
| B | 3〜5分 | 堅実な踏破者 |
| C | 5〜8分 | 辛勝の脱出者 |
| D | 8分以上 | 瀕死の帰還者 |
