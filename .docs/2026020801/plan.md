# Lintエラー修正 実装計画

## 作成日
- 2026-02-08

## 目的
- `npm run lint` で報告されるエラーを全件解消し、今後の開発で lint が継続的に通る状態を作る。
- `react-hooks/exhaustive-deps` の warning も解消し、Hook依存関係を明示化する。

## スコープ
- 対象: `src/` 配下の TypeScript / TSX
- 非対象: 機能追加、UI仕様変更、大規模リファクタ

## 方針
1. lint結果をカテゴリ化して優先度順に対応
2. 振る舞いを変えない修正を優先
3. ルール由来で過剰なノイズがある場合は、最小限のESLint設定調整を検討
4. 最終的に `npm run lint` を再実行して0エラーを確認

## 実施フェーズ
1. 計画・仕様・タスク定義
2. エラー修正実装
3. warning修正実装（`react-hooks/exhaustive-deps`）
4. 検証（lint再実行）
5. 変更差分の要約

## システムテスト影響範囲
- 影響画面: `IpnePage`（ゲーム進行、戦闘、レベルアップ、ゲームオーバー/リトライ）
- 重点確認:
  1. タイトル→職業選択→プロローグ→ゲーム開始が正常遷移すること
  2. 移動・攻撃・壁発見・被ダメージが従来通り動作すること
  3. 敵撃破時のレベルアップポイント加算とモーダル開閉が正しいこと
  4. ゲームオーバー後のリトライでマップ再初期化が正しいこと
  5. クリア時のタイマー停止・評価計算・記録保存が正しいこと

## 完了条件
- `npm run lint` がエラー0で終了する
- `npm run lint` で warning 0 で終了する
- 変更内容が `.docs/2026020801/tasks.md` の完了状態と整合する
