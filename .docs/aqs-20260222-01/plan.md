# Agile Quiz Sugoroku - クイズタグ付けジャンル分け 実装計画

## 概要

380 問超のクイズに横断的なジャンルタグを付与し、将来的なフィルタリングや研修シーン別出題を可能にする。

## 方針

- `Question` インターフェースに `tags?: string[]` を追加（optional で後方互換性維持）
- タグマスタ定義ファイル `tag-master.ts` を新規作成（16 ジャンル）
- 全 8 JSON ファイルの全問題にタグを付与
- バリデーション関数を更新しタグの整合性を保証
- 既存のゲームロジック・コンポーネント・テストは一切変更しない

## フェーズ

1. ドキュメント作成（plan.md / spec.md / tasks.md）
2. 型定義・タグマスタ・バリデーション更新
3. 全 380 問にタグ付与
4. テスト追加・検証

## 検証方法

- `npm test` — 既存 + 新規テスト全通過
- `npm run build` — ビルド成功
- 16 タグ全てが最低 1 問に使われていること
