# Air Hockey ブラッシュアップ ナラティブフロー v3

## コンセプト: 「見て、感じて、また来たくなる」

v2では「触って気持ちいい + 次のステージに進みたい」を実現した。
v3では「視覚的な没入感 + キャラへの愛着 + 友達と共有」を加える。

## なぜこの順番なのか

1. **Phase 0 が最初**: ビジュアルを作る前にキャラの設定が固まっていないと、立ち絵の雰囲気や背景の選定がブレる。30分〜1時間のドキュメント作業で後続フェーズの品質が上がる。

2. **Phase 1 が本命**: v3 の核心。テキストだけのストーリーに「絵の力」を加えることで、没入感が劇的に向上する。**全ユーザーが通るストーリーモードの体験を底上げする**のが最もコスパが高い。

3. **Phase 2 と Phase 3 は並行可能**: キャラ図鑑はストーリーの延長線、2人対戦は独立した新モード。依存関係がないので同時進行できる。

4. **Phase 3 が後ろ**: 2人対戦は技術的にも仕様的にも検討事項が多い。ストーリー演出が完成した状態で「2人でストーリーを進める」モードの可能性も見えてくる。

## v2 → v3 の差分まとめ

| 要素 | v2（実装済み） | v3（今回の計画） |
|------|---------------|----------------|
| ダイアログ背景 | 暗転のみ | 場面別背景画像 |
| キャラ表示 | 64-128px アイコン | 立ち絵（表情差分付き） |
| VS画面 | 名前+アイコン | 全身イラスト+エフェクト |
| ステージ開始 | 直接ダイアログ | チャプタータイトルカード→ダイアログ |
| クリア演出 | テキスト「TO BE CONTINUED」 | カットインイラスト + 桜エフェクト |
| キャラ情報 | ダイアログ内のみ | キャラクター図鑑（アンロック式） |
| マルチプレイ | なし | 2P対戦（キーボード/タッチ）+ 協力プレイ |
| 世界観 | 断片的 | 設定ドキュメント + ゲーム内表現 |
