# Air Hockey（エアホッケー）

## 概要

CPU対戦型のエアホッケーゲーム。
難易度・フィールド・勝利スコアを選択して対戦する「フリー対戦」モードと、
キャラクター付きの3ステージを順にクリアする「ストーリーモード」を搭載。
物理演算によるリアルなパック挙動とAI戦略を実装。
レスポンシブデザインにより、スマートフォンからデスクトップまで快適にプレイ可能。

## 既知の問題

### 画像アセット

- **立ち絵の画風がキャラ間で統一されていない**: AI画像生成（Nanobanana2）の制約により、生成ごとに画風のブレがある。特に同一キャラの normal/happy 間でも顔つきが異なる場合がある
- **128×128 アイコンと立ち絵の整合性**: 既存アイコン（`/assets/characters/`）と新規立ち絵（`/assets/portraits/`）で見た目の印象が異なるキャラがいる
- **VS用画像の品質**: 現在は透過済み立ち絵からのトリミングで代用しており、専用イラストではない
- **勝利カットインが第1章分のみ**: `victory-ch1.png` のみ存在。第2章以降は未作成
- **透過処理のアーティファクト**: クロマキー方式（グリーンバック/ブルーバック→透過変換）のため、輪郭部分に微細な色残りが発生している箇所がある
- **ユウのVS用画像が未作成**: `/assets/vs/yuu-vs.png` が存在しない（第2章で登場予定のため未作成）

## ゲームモード

### フリー対戦

難易度・フィールド・勝利スコアを自由に選択して CPU と対戦するモード。

### ストーリーモード

エアホッケー部の新入りとして、部員たちと順に対戦していく3ステージ構成。

| ステージ | 対戦相手 | 難易度 | フィールド | 勝利スコア |
|---------|---------|--------|----------|-----------|
| 1-1 はじめの一打 | ヒロ | Easy | Original | 3点 |
| 1-2 テクニカルな壁 | ミサキ | Normal | Wide | 3点 |
| 1-3 部長の壁 | タクマ | Hard | Pillars | 5点 |

## クイックリファレンス

### 操作方法

- **マウス / タッチ**: マレット（パドル）を操作
- **キーボード**: WASD / 矢印キーでマレット移動
- **ポーズ**: Escape / P キー

### 難易度

| 難易度 | CPU の特徴 |
|--------|-----------|
| Easy | ブレが大きく低速 |
| Normal | 予測＋積極的な動き |
| Hard | 壁反射予測＋高精度ポジショニング |

### 勝利スコア

3 / 7 / 15 点から選択
