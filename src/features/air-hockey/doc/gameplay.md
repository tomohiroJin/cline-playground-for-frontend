# ゲームプレイガイド

## 操作方法

### 1P モード（CPU 対戦）

- **マウス / タッチ**: マレット（パドル）を操作
- **キーボード**: WASD / 矢印キーでマレット移動（速度はマウスより控えめに調整）
- **ポーズ**: Escape / P キー、またはスコアボードのポーズボタン

### 2P モード（ローカル対戦）

| プレイヤー | 操作 | 移動範囲 |
|-----------|------|---------|
| 1P | 矢印キー（↑↓←→）/ マウス / タッチ | 画面下半分 |
| 2P | WASD キー / タッチ | 画面上半分 |

- **ポーズ**: Escape / P（どちらのプレイヤーからも可能）
- 2P / ペアマッチ時は P1 のキーボード操作が矢印キーのみに制限（WASD は P2 専用）

### ペアマッチ（2v2）モード

| スロット | 役割 | 操作 | 移動範囲 |
|---------|------|------|---------|
| P1 | プレイヤー | 矢印キー / マウス / タッチ | 下半分全体 |
| P2 | パートナー | CPU（AI）または 人間（WASD / タッチ / 🎮 Gamepad 1） | 下半分全体 |
| P3 | 敵 1 | CPU（AI）または 人間（🎮 Gamepad 2） | 上半分全体 |
| P4 | 敵 2 | CPU（AI）または 人間（🎮 Gamepad 3） | 上半分全体 |

- P2 は TeamSetupScreen で CPU/人間を切り替え可能
- P3/P4 はゲームパッド接続時のみ人間操作に切り替え可能（P3: 2本以上、P4: 3本以上接続が必要）
- ゲームパッドのアナログスティックで操作（デッドゾーン 0.15、非線形カーブ適用）
- 味方同士は同じ半面全体を自由に移動可能（マレット間衝突判定あり）
- ゲーム中にメニューボタン → 確認ダイアログ表示（2v2 モードのみ）

## ゲームルール

### 基本ルール

- 難易度: Easy / Normal / Hard（CPU速度・予測精度が異なる）
- 勝利スコア: 3 / 7 / 15 点から選択
- 得点後のサーブ: **失点した側からパックが出発**（失点側がサーブ権を持つ）
- 初期サーブ: ランダムに上下どちらかへ発射

### フィーバーモード

最後のゴールから15秒経過で追加パック投入（最大3つ、虹色演出）。

### コンボシステム

連続得点でコンボ演出が発生:
- x3: オレンジ演出
- x5+: レインボー演出

### カムバックメカニクス

スコア差3以上で劣勢側に微小なアシスト（マレット拡大・ゴール縮小）。

## フィールド・ステージ

| ステージ | 特徴 |
|---------|------|
| Original | 障害物なしの標準フィールド |
| Wide | ゴールが広い |
| Pillars | 固定障害物あり |
| Zigzag | ジグザグ配置の固定障害物 |
| Fortress | **破壊可能障害物**（HP3、パック衝突で減少、破壊後5秒で復活） |
| Bastion | **破壊可能障害物**（上下段の壁＋中央ブロック、HP3、5秒で復活） |

※ Pillars 以降は勝利数によるアンロック制（詳細は [features.md](./features.md#フィールドアイテムアンロック) 参照）

## アイテム

| アイテム | 効果 |
|---------|------|
| Split | パック3分裂 |
| Speed | 速度変化 |
| Invisible | パック透明化 |
| Shield | ゴール前にバリア（1回防御で消滅） |
| Magnet | マレットがパックを引き寄せる（5秒） |
| Big | マレットが1.5倍に拡大（8秒） |

※ Shield / Magnet / Big はアンロック制（詳細は [features.md](./features.md#フィールドアイテムアンロック) 参照）
