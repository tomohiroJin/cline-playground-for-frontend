# 機能詳細

## レスポンシブデザイン

- 内部解像度 450x900 固定、CSS スケーリングで画面サイズに自動フィット
- アスペクト比 1:2 を維持し、小画面では幅いっぱい、大画面では max-width で制限

## カウントダウン演出

- ゲーム開始時に「3 → 2 → 1 → GO!」のカウントダウン表示
- カウントダウン中はパック・マレットの移動停止

## 画面シェイク

- ゴール時に強シェイク、強打ヒット時に弱シェイク（CSS transform で実装）

## パック速度ビジュアル

速度に応じてパックの色・トレイル長・グローが変化:
- 通常: 白色
- 高速: 黄色
- 超高速: 赤色＋スピードライン

## サウンド

- **BGM**: Web Audio API でゲーム中にループ BGM を動的生成（フィーバー時テンポアップ）
- **SE**: ヒット音が速度に応じて変化、ゴール時のファンファーレ/下降音

## 音量設定

- BGM / SE を個別にスライダーで調整（0〜100%）
- ミュートボタン、設定は localStorage に永続化

## ポーズ機能

- ゲーム中いつでも一時停止可能
- ポーズ中: 「Resume」「Restart」「Back to Menu」を選択

## リザルト画面

- 勝利時: 紙吹雪エフェクト
- 試合統計のカウントアップアニメーション表示（ヒット数、最高パック速度、アイテム取得数、セーブ数、試合時間）
- MVP スタッツのハイライト表示
- 「REPLAY」ボタンで同条件で即再戦
- 新規実績解除時のポップアップ表示
- 難易度オートアジャスト提案の表示

## 実績システム

| 実績 | 条件 |
|------|------|
| 初勝利 | 初めて CPU に勝利 |
| パーフェクト | 無失点で勝利 |
| 3連勝 | 3回連続で勝利 |
| 5連勝 | 5回連続で勝利 |
| ハードモード制覇 | Hard で勝利 |
| フィールドマスター | 全フィールドで勝利 |
| 大逆転 | 3点差以上から逆転勝利 |
| スピードデーモン | パック速度 15 以上を記録 |
| アイテムコレクター | 全種類のアイテムを使用 |

- タイトル画面から実績一覧を確認可能

## 難易度オートアジャスト

- 3連敗で難易度を下げる提案、3連勝で上げる提案を自動表示

## フィールド/アイテムアンロック

初期利用可能: Original, Wide フィールド + Split, Speed, Invisible アイテム

| アンロック対象 | 条件 |
|--------------|------|
| Pillars | 3勝 |
| Zigzag | 5勝 |
| Fortress | 8勝 |
| Bastion | 12勝 |
| Shield | 3勝（Normal以上） |
| Magnet | Hard で勝利 |
| Big | 10勝 |

- アンロック状態は localStorage に永続化

## デイリーチャレンジ

毎日異なる特殊ルール（日付ベースのシード値で全ユーザー共通条件）。クリア結果を localStorage に保存。

| パターン | 特殊ルール | 勝利スコア |
|---------|-----------|-----------|
| スピードラッシュ | パックが高速 | 3点 |
| タイニーゴール | ゴールが狭い | 3点 |
| アイテムフィーバー | アイテム大量出現 | 7点 |
| ビッグマレット | マレットが巨大 | 3点 |
| サバイバル | アイテムなし | 7点 |
| カオスマッチ | パック高速＋アイテム大量 | 3点 |
| マスターチャレンジ | ゴールが狭い | 7点 |

## チュートリアル

初回プレイ時に4ステップのインタラクティブチュートリアルを表示:
1. マレットを動かしてみよう（マウス/タッチ操作）
2. パックを打ち返そう
3. ゴールに入れよう
4. アイテムを使おう

- スキップ可能、完了フラグは localStorage に保存
- タイトル画面の `?` ボタンからいつでも再確認可能

## ゲーム中ヘルプ

- 5秒間無操作で自動ヘルプを表示
- 全6種アイテムのアイコン・名前・説明、現在のフィールド特徴を表示
- タップで閉じてゲーム再開

## 画面トランジション

- menu → game → result の各遷移にフェードトランジション
