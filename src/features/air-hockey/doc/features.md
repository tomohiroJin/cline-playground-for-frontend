# 機能詳細

## レスポンシブデザイン

- 内部解像度 600x1200 固定、CSS スケーリングで画面サイズに自動フィット
- アスペクト比 1:2 を維持し、小画面では幅いっぱい、大画面では max-width で制限

## カウントダウン演出

- ゲーム開始時に「3 → 2 → 1 → GO!」のカウントダウン表示
- カウントダウン中はパック・マレットの移動停止

## 画面シェイク

- ゴール時に強シェイク、強打ヒット時に弱シェイク（CSS transform で実装）

## パック速度ビジュアル

速度に応じてパックの色・トレイル長・グローが変化:
- 通常: 白色
- 高速: 黄色
- 超高速: 赤色＋スピードライン

## サウンド

- **BGM**: Web Audio API でゲーム中にループ BGM を動的生成（フィーバー時テンポアップ）
- **SE**: ヒット音が速度に応じて変化、ゴール時のファンファーレ/下降音

## 音量設定

- BGM / SE を個別にスライダーで調整（0〜100%）
- ミュートボタン、設定は localStorage に永続化

## ポーズ機能

- ゲーム中いつでも一時停止可能
- ポーズ中: 「Resume」「Restart」「Back to Menu」を選択

## リザルト画面

- 勝利時: 紙吹雪エフェクト
- 試合統計のカウントアップアニメーション表示（ヒット数、最高パック速度、アイテム取得数、セーブ数、試合時間）
- MVP スタッツのハイライト表示
- 「REPLAY」ボタンで同条件で即再戦
- 新規実績解除時のポップアップ表示
- 難易度オートアジャスト提案の表示

## 実績システム

| 実績 | 条件 |
|------|------|
| 初勝利 | 初めて CPU に勝利 |
| パーフェクト | 無失点で勝利 |
| 3連勝 | 3回連続で勝利 |
| 5連勝 | 5回連続で勝利 |
| ハードモード制覇 | Hard で勝利 |
| フィールドマスター | 全フィールドで勝利 |
| 大逆転 | 3点差以上から逆転勝利 |
| スピードデーモン | パック速度 15 以上を記録 |
| アイテムコレクター | 全種類のアイテムを使用 |

- タイトル画面から実績一覧を確認可能

## 難易度オートアジャスト

- 3連敗で難易度を下げる提案、3連勝で上げる提案を自動表示

## フィールド/アイテムアンロック

初期利用可能: Original, Wide フィールド + Split, Speed, Invisible アイテム

| アンロック対象 | 条件 |
|--------------|------|
| Pillars | 3勝 |
| Zigzag | 5勝 |
| Fortress | 8勝 |
| Bastion | 12勝 |
| Shield | 3勝（Normal以上） |
| Magnet | Hard で勝利 |
| Big | 10勝 |

- アンロック状態は localStorage に永続化

## デイリーチャレンジ

毎日異なる特殊ルール（日付ベースのシード値で全ユーザー共通条件）。クリア結果を localStorage に保存。

| パターン | 特殊ルール | 勝利スコア |
|---------|-----------|-----------|
| スピードラッシュ | パックが高速 | 3点 |
| タイニーゴール | ゴールが狭い | 3点 |
| アイテムフィーバー | アイテム大量出現 | 7点 |
| ビッグマレット | マレットが巨大 | 3点 |
| サバイバル | アイテムなし | 7点 |
| カオスマッチ | パック高速＋アイテム大量 | 3点 |
| マスターチャレンジ | ゴールが狭い | 7点 |

## チュートリアル

初回プレイ時に4ステップのインタラクティブチュートリアルを表示:
1. マレットを動かしてみよう（マウス/タッチ操作）
2. パックを打ち返そう
3. ゴールに入れよう
4. アイテムを使おう

- スキップ可能、完了フラグは localStorage に保存
- タイトル画面の `?` ボタンからいつでも再確認可能

## ゲーム中ヘルプ

- タイトル画面の「？」ボタンから手動で表示
- 全6種アイテムのアイコン・名前・説明、現在のフィールド特徴を表示
- タップで閉じてゲーム再開
- ※ ゲーム中の自動ヘルプ表示は無効化済み

## 2P 対戦モード

- タイトル画面の「2P 対戦」ボタンからローカル対戦を開始
- キャラクター選択画面で 1P / 2P が好きなキャラを選択可能（ミラーマッチ対応）
- フィールド・勝利スコアも自由に設定可能

### 操作方法

| プレイヤー | キーボード |
|-----------|-----------|
| 1P | 矢印キー（↑↓←→） |
| 2P | WASD キー |

- ポーズ: Escape / P（共通）

### 2P 対戦の特徴

- CPU AI を使わず、2 人のプレイヤーが同時にマレットを操作
- 2P 対戦では実績・アンロック・ストーリー進行は無効
- リザルト画面で「1P Win!」/「2P Win!」を表示
- 「キャラ選択に戻る」ボタンで連戦が可能

## ペアマッチ（2v2）モード

- タイトル画面の「ペアマッチ」ボタンから開始
- チーム構成: 人+CPU vs CPU+CPU（デフォルト）/ 人+人 vs CPU+CPU
- P2 は CPU/人間をセグメントコントロールで切り替え可能
- P3/P4 はキャラクター選択可能（アンロック済みキャラから選択）
- チーム1（青）/チーム2（赤）の色分けで視覚的に区別

### ペアマッチの特徴

- 味方同士は同じ半面全体を自由に移動可能（上下 2 分割ゾーン）
- ゴールサイズが拡大（通常フィールド: 240px、Wide: 280px）
- マレット間衝突判定あり（重なり防止、ゾーン制約の再適用）
- パックスタック脱出（0.5 秒間低速でフィールド中央方向に射出）
- キャラ別 AI プロファイル（下表参照）
- ally CPU には味方補正（aggressiveness 上限 0.5 で守備的に行動）

### キャラクター別 AI プロファイル（実装済み）

| キャラ | スタイル | aggressiveness | 揺さぶり | 適応度 | 特徴 |
|--------|---------|---------------|---------|--------|------|
| ヒロ | ストレートシューター | 0.7（攻撃的） | なし | 低 | 直線的でシンプル、前に出る |
| ミサキ | テクニシャン | 0.5（バランス） | 大（40px/2秒周期） | 中低 | サイドへの揺さぶりが多い |
| タクマ | パワーバウンサー | 0.2（守備的） | なし | 低 | ゴール前に構える守備重視 |
| ユウ | アナライザー | 0.4（やや守備的） | 控えめ（20px/3秒） | **高（0.8）** | スコア差に応じて強くなる |
| ルーキー | ビギナー | 0.3（消極的） | なし | なし | 動きが遅く反応が鈍い |
| レギュラー | オールラウンダー | 0.5（バランス） | わずか（10px/4秒） | 低 | バランス型 |
| エース | エリート | 0.6（やや攻撃的） | 正確（15px/2.5秒） | 中 | 高精度な予測と速い反応 |

> **将来の改善**: 現在の個性は `aggressiveness`（前後ポジション）と `lateralOscillation`（揺さぶり）が中心。
> 守備位置パターン、打ち返し角度のクセ、リアクション速度、連携 AI は後続フェーズで対応予定。
- VsScreen で 4 キャラ表示 + チームラベル + P2 操作タイプラベル
- ResultScreen で 4 体立ち絵 + チーム間⚡区切り
- 「同じ設定でリプレイ」「チーム設定に戻る」の導線
- `prefers-reduced-motion` 対応（アニメーションスキップ）

## 画面トランジション

- menu → game → result の各遷移にフェードトランジション（フリー対戦）
- menu → characterSelect → vsScreen → game → result（2P 対戦）
- menu → teamSetup → vsScreen → game → result（ペアマッチ）
- menu → stageSelect → preDialogue → vsScreen → game → postDialogue → result（ストーリーモード）

## ストーリーモード

### 画面構成

- **ステージ選択画面**: 解放済みステージをカード形式で表示、クリア済みにチェックマーク
- **ダイアログオーバーレイ**: キャラアイコン＋名前＋セリフ（1文字ずつ表示、スキップ可能）
- **VS 画面**: 対決演出（2秒自動遷移、フェードイン/アウト）

### キャラクターリアクション

- ゴール時に得点側・失点側キャラの吹き出しを表示
- 勝利/敗北時に専用リアクション

### バランス調整

各ステージに個別の AI 設定が適用される。詳細は [ストーリーモード](./story-mode.md) を参照。

### ヒットストップ・スローモーション

- 強打（パック速度 > 8）時に3フレームのヒットストップ＋衝撃波演出
- ゴール時に 400ms のスローモーション＋ビネット効果
