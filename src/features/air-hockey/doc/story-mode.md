# ストーリーモード

## 概要

エアホッケー部に入部した主人公「アキラ」が、部員たちと対戦しながら成長していくモード。
第1章は3ステージ構成で、各ステージをクリアすると次が解放される。

## 画面遷移

```
タイトル → ステージ選択 → 試合前ダイアログ → VS画面 → ゲーム → 試合後ダイアログ → リザルト
```

> **注意**: 2P 対戦モード（`gameMode === '2p-local'`）では、ストーリー進行・実績判定・アンロック処理は無効化されます。

## バランス設定

各ステージには個別の AI バランス設定があり、段階的に難易度が上昇する。

### ステージ 1-1: はじめの一打（初心者が2-3回で勝てる）

- CPU 速度: 1.2（デフォルト Easy の 1.5 より遅い）
- 予測係数: 0.5（ほぼ予測なし）
- ウォブル: 40px（大きなブレ）
- スキップ率: 10%（CPU がときどき反応しない）
- 中央寄せ: 80%（ターゲットを大きく中央に寄せる）

### ステージ 1-2: テクニカルな壁（アイテム活用で勝てる）

- CPU 速度: 3.0
- 予測係数: 4
- アイテム出現間隔: 4秒（デフォルト6秒より短い）
- カムバック閾値: スコア差2（デフォルト3より早く発動）
- カムバックマレットボーナス: 15%

### ステージ 1-3: 部長の壁（苦戦するが練習すれば勝てる）

- CPU 速度: 5.0（デフォルト Hard の 6 より少し遅い）
- 予測係数: 10
- 壁バウンス予測: あり
- カムバック閾値: スコア差2
- カムバックゴール縮小: 15%

## AI 振る舞い設定（AiBehaviorConfig）

AI の振る舞いは `AiBehaviorConfig` 型で制御される。
フリー対戦では `AI_BEHAVIOR_PRESETS` のプリセットを使用し、
ストーリーモードでは `getStoryStageBalance()` でステージごとのカスタム設定を使用する。

### パラメータ一覧

| パラメータ | 説明 | Easy | Normal | Hard |
|-----------|------|------|--------|------|
| maxSpeed | CPU 最大移動速度 | 1.5 | 3.5 | 6.0 |
| predictionFactor | パック軌道予測係数 | 1 | 6 | 12 |
| wobble | 狙いのブレ幅(px) | 30 | 0 | 0 |
| skipRate | フレームスキップ率 | 5% | 0% | 0% |
| centerWeight | 中央寄せ重み | 70% | 0% | 0% |
| wallBounce | 壁バウンス予測 | なし | なし | あり |

## カムバック補正

スコア差が閾値以上になると、劣勢側にボーナスが付与される。

| パラメータ | デフォルト | 説明 |
|-----------|----------|------|
| THRESHOLD | 3 | 発動するスコア差 |
| MALLET_BONUS | 10% | マレットサイズ増加 |
| GOAL_REDUCTION | 10% | 優勢側ゴール縮小 |

## データの永続化

- ストーリー進行（クリア済みステージ）は `localStorage` に保存
- キー: `ah_story_progress`
- リセット機能あり（ステージ選択画面から実行可能）
