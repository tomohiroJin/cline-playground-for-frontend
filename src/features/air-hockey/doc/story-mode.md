# ストーリーモード

## 概要

エアホッケー部に入部した主人公「アキラ」が、部員たちと対戦しながら成長していくモード。
現在第2章まで実装済みで、計7ステージ構成。各ステージをクリアすると次が解放される。

## 画面遷移

```
タイトル → ステージ選択 → [チャプタータイトル →] 試合前ダイアログ → VS画面 → ゲーム → 試合後ダイアログ → [勝利カットイン →] リザルト
```

> **注意**: 2P 対戦モード（`gameMode === '2p-local'`）では、ストーリー進行・実績判定・アンロック処理は無効化されます。

## バランス設定

各ステージには個別の AI バランス設定があり、段階的に難易度が上昇する。

### 第1章 バランス設定

#### ステージ 1-1: はじめの一打

- CPU 速度: 1.6 / 予測係数: 0.7 / ウォブル: 53px / スキップ率: 10%
- キャラ AI: ヒロ（aggressiveness: 0.7, attacker）

#### ステージ 1-2: テクニカルな壁

- CPU 速度: 4.0 / 予測係数: 5.3 / ウォブル: 13px
- アイテム出現間隔: 4秒（デフォルト6秒より短い）
- カムバック閾値: スコア差2 / マレットボーナス: 15%
- キャラ AI: ミサキ（deflectionBias: 0.5, wide defense）

#### ステージ 1-3: 部長の壁

- CPU 速度: 6.7 / 予測係数: 13.3 / 壁バウンス予測: あり
- カムバック閾値: スコア差2 / ゴール縮小: 15%
- キャラ AI: タクマ（aggressiveness: 0.2, defender, reaction: 30ms）

### 第2章 バランス設定

## AI 振る舞い設定（AiBehaviorConfig）

AI の振る舞いは `AiBehaviorConfig` 型で制御される。
フリー対戦では `AI_BEHAVIOR_PRESETS` のプリセットを使用し、
ストーリーモードでは `getStoryStageBalance()` でステージごとのカスタム設定を使用する。

### パラメータ一覧

| パラメータ | 説明 | Easy | Normal | Hard |
|-----------|------|------|--------|------|
| maxSpeed | CPU 最大移動速度 | 2.0 | 4.7 | 8.0 |
| predictionFactor | パック軌道予測係数 | 1.3 | 8 | 16 |
| wobble | 狙いのブレ幅(px) | 40 | 0 | 0 |
| skipRate | フレームスキップ率 | 5% | 0% | 0% |
| centerWeight | 中央寄せ重み | 70% | 0% | 0% |
| wallBounce | 壁バウンス予測 | なし | なし | あり |

## カムバック補正

スコア差が閾値以上になると、劣勢側にボーナスが付与される。

| パラメータ | デフォルト | 説明 |
|-----------|----------|------|
| THRESHOLD | 3 | 発動するスコア差 |
| MALLET_BONUS | 10% | マレットサイズ増加 |
| GOAL_REDUCTION | 10% | 優勢側ゴール縮小 |
