## キャラクターリアクション（ゲーム中）

クイズ画面にネコ・イヌ・ウサギの3体が常時表示され、状況に応じたコメントを表示:

| 状況 | タイミング | コメント例 |
|------|-----------|-----------|
| 待機中 | 回答前・余裕あり | 「考えるにゃ...」「どれかなワン？」「うーんぴょん」 |
| 残り7秒 | timer <= 7 | 「そろそろ決めるにゃ」「時間あるワン」「考えてぴょん」 |
| 残り5秒（ヒント） | timer <= 5 | 得意分野キャラが問題タグに関連するヒントを表示 |
| 残り3秒 | timer <= 3 | 「急ぐにゃ！」「時間がないワン！」「早くぴょん！」 |
| 正解 | 回答後 | 「にゃいす！」「その通りワン！」「正解ぴょん！」 |
| 不正解 | 回答後 | 「ドンマイにゃ」「次がんばるワン」「惜しいぴょん」 |
| コンボ3+ | 回答後 | 「ノッてるにゃ！」「すごいワン！」「絶好調ぴょん！」 |
| 緊急対応 | イベント | 「障害発生にゃ！」「落ち着くワン！」「冷静にぴょん！」 |

得意分野（ネコ: 設計・プログラミング、イヌ: スクラム・バックログ、ウサギ: テスト・CI/CD）に応じたヒントを提供。

## キャラクターナラティブ

スプリント開始画面・振り返り画面でキャラクターが状況に応じた会話を表示:

- Sprint 1 開始時: 意気込みのセリフ
- Sprint 2+ 開始時（成績良好）: 応援のセリフ
- Sprint 2+ 開始時（負債多）: 警告のセリフ
- 振り返り（高正答率）: 称賛のセリフ
- 振り返り（低正答率）: 励ましのセリフ

各パターン3キャラ × 3バリエーション = 45件のセリフを収録。

## ストーリー機能

スプリント間にノベルゲーム風のストーリーが挿入され、5人のキャラクターによるチームの成長物語を体験できます。

- 全8話構成（スプリント数に応じて表示話数が変化）
- キャラクターの台詞とナレーションによる演出
- スプリント背景画像のフェードイン表示
- Enter/Space で進行、Escape でスキップ可能

### エンディングストーリー

最終スプリント後に、共通パート + チームタイプ別エピローグが再生されます。
